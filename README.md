# 📌 Project Overview

This project focuses on Spotify User Behavior Analysis and Churn Prediction using Exploratory Data Analysis (EDA).
The goal is to analyze user listening behavior and identify patterns that contribute to user churn.

Understanding churn helps businesses:

Improve user retention
Increase engagement
Optimize subscription strategies
# 📊 Dataset

File: spotify_churn_dataset.csv

The dataset contains user behavior information such as:

User demographics
Subscription type
Listening time
Skip rate
Engagement metrics
Churn status
